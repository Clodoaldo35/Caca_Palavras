Como criar os ícones para o PWA:

  1. Faça uma captura de tela do ícone acima (512x512)
  2. Utilize uma ferramenta online como iLoveIMG para redimensionar a imagem para todos os tamanhos necessários:
    • 72x72
    • 96x96
    • 128x128
    • 144x144
    • 152x152
    • 192x192
    • 384x384
    • 512x512
  3. Salve cada imagem com o nome correspondente na pasta icons (por exemplo: icon-72x72.png)
  4. Certifique-se de que todos os arquivos estão no formato PNG
